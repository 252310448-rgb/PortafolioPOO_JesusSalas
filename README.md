## Descripción General
Este portafolio contiene los proyectos integradores desarrollados a lo largo del semestre en la carrera de Ingeniería Informática. El objetivo es demostrar el dominio de la Programación Orientada a Objetos (POO), la correcta estructuración de código, el control de versiones y el despliegue local de aplicaciones.

### Tecnologías Utilizadas
* **Lenguajes:** PHP, Python, HTML, CSS
* **Herramientas y Servidores:** XAMPP (Apache), GitHub

### Competencias Desarrolladas
* Aplicación de los pilares de POO (Herencia, Encapsulamiento, Polimorfismo, Abstracción).
* Gestión de arreglos unidimensionales y manipulación de archivos de texto (bitácoras).
* Implementación de modularidad mediante métodos y control de lógica de consumo energético.
